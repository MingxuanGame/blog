# blog (blog.mxgame.top)

此仓库为我个人博客页面源代码，部署仓库为 [MingxuanAWA/blog](https://github.com/MingxuanAWA/blog)

由 Hexo 驱动，主题为 [theme-ad](https://github.com/dongyuanxin/theme-ad)

通过 GitHub Action，推送时自动部署到远程仓库
